Convera
A social media web app inspired by reddit

Using Laravel, Google OAuth, Laravel Breeze

Libraries used : Bootstrap, Fontawesome, Tailwind

Mostly using native CSS & Javascript

Features :
Google OAuth
Sessions
CRUD for Profiles
CRUD for Post
Follow Users, Communities
Comments & Likes
Saved Post
Communities (WIP)
Notifications (WIP)
Messages (WIP)
Tags (WIP)
Settings (WIP)
